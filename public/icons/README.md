# PWA 图标说明

请在此目录放置以下尺寸的 PNG 图标：

- `icon-192x192.png` - 192x192 像素
- `icon-512x512.png` - 512x512 像素

## 快速生成图标

你可以使用以下工具生成 PWA 图标：

1. **在线工具**:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. **设计建议**:
   - 使用简洁的图标设计
   - 主色调建议使用 #3B82F6（科技蓝）
   - 背景可以是透明或纯色
   - 确保图标在小尺寸下清晰可辨

## 临时方案

在正式图标准备好之前，你可以：
1. 使用 emoji 作为临时图标
2. 使用纯色背景 + 文字
3. 从 https://www.flaticon.com/ 下载免费图标
